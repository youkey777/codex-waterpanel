# ウォーターサーバー操作パネル デモ

## 概要
- ウォーターサーバーの操作パネル挙動をブラウザで確認するデモページです
- 画像の上に透明ボタンを重ねて、各種挙動をシミュレーションします

## 使い方
- `index.html` をブラウザで開いて操作してください
- 上部が操作パネル、下部が取水口のアニメーションです

## 操作仕様の要点
- 量指定: `+/-` で10ml単位、長押しで連続増減（10〜5000ml）
- 冷水/常温水: ロック不要、押下中は取水、量指定時は自動取水
- 温水/白湯: 3秒長押しでロック解除後に取水可能（解除後5秒で再ロック）
- 粉ミルク: ロック解除 + 量指定必須、60%温水 → 40%冷水の2段階
- ECO: 単押しでオン/オフ、長押しでキャンセル

## 画像について
- `操作パネル通常.png` と `取水口.png` を使用しています
- ボタン位置は初期値のため、実画像に合わせて調整してください
